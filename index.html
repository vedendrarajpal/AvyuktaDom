Trunking Architecture

Choose your line, choose your math.

Two ways in, two ways to host — then a dialing mode that decides how much of your agent hour is actually spent talking.

Domestic Calling Possibilities

Pick the CLI that matches the campaign.

Three routes to a domestic dial tone — each with a different pickup ratio, a different rulebook.

Option 1

Hosted Dialer with SIP (079/074/080)

Mobile-Like CLI

  • Only for Transactional campaigns / Opt-in Leads
  • Unlimited Inbound and Outbound Calls (FUP*)
  • 1–2 DIDs per channel included
  • Additional DID ₹400
  • 1 month advance + 2 months security
Option 2

Hosted Dialer with SIP 94****

Mobile CLI

  • Mobile Like CLI for Better Pick Up Ratio
  • Only for Transactional campaigns / Opt-in Leads
  • 3,500 mins/channel/month FUP
  • Additional DID ₹700/month
  • 6-month lock-in, quarterly billing
GSM Route

Hosted Dialer with Client GSM

VoLTE / Non-VoLTE SIM

  • For both Transactional and promotional campaigns
  • Ideal for cold calling campaigns
  • GSM SIMs needs to be procured by client
  • Display number = the SIM's own number
  • Monthly billing, no lock-in or security

Pro tip: auto-dialing needs 1.5–2× the agent headcount to keep calls flowing back-to-back.
